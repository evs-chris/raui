This decorator takes an input and sets it up for display and input of numeric values. It allows specifying number of whole and decimal places, including disallowing decimals entirely. ### Plugin options All options are optional. * `name: string ='numeric'` - the name to use when registering the decorator as a plugin ### Arguments This decorator takes a string or an options object. Passing a string is equivalent to `{ bind: string }`. * `bind: keypath` - a keypath to bind the string version of the number, without prefix or suffix * `number: keypath` - a keypath to use to bind the input value as a number * `optional: boolean = true` - if no number is supplied for a non-optional input, the value with be `0`. For optional inputs, the value will be an empty string and the number will be undefined. * `lazy: boolean = true` - whether or not to immediately update the bound values * `whole: number` - an optional number of whole digits to allow in the input * `decimal: number` - an optional number of decimal digits to all in the input * `preferInteger: boolean = true` - automatically drop `.0+` from the input. * `prefix: string` - optionally prefix the numeric input with a string. Input starts after the prefix and cursor placement is handled automatically. * `suffix: string` - optionally suffix the numeric input with a string. Input starts before the suffix and cursor placement is handled automatically.
